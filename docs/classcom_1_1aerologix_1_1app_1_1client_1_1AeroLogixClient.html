Detailed Description

This class is the main entry point for the AeroLogix client application.

The

AeroLogixClient

class is the main entry point for the AeroLogix client application. It is responsible for initializing the REST client, setting up the web target, and providing methods to interact with the server.

This class follows the Singleton design pattern to ensure only one instance is created.
