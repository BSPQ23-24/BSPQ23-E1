Detailed Description

Class of the panel for the form to register a new booking.

Class of the panel for the form to register a new booking.

It is a JPanel that contains multiple JTextFields to manually type in the Passenger information and a JDatePicker for the passenger birth date. Additionally, it contains a JComboBox to select the airline for the booking.
