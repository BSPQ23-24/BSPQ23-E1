Detailed Description

Class of the panel for the form to modify an existent booking.

Class of the panel for the form to modify an existent booking.

It is a JPanel that contains multiple JTextFields to manually modify the Passenger information and a JDatePicker for the passenger birth date. Additionally, it contains a JComboBox to select the airline for the booking.
